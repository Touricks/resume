## Features

- **Listing Management**: Upload, delete, search, and reserve accommodation listings
- **Token-Based Authentication**: Secure user authentication system
- **Database Design**: PostgreSQL schema with Spring Data JPA repositories
- **Cloud Storage**: Google Cloud Storage integration for image files
- **Location-Based Search**: Geocoding service integration for location-aware queries
- **User-Friendly Interface**: Built with React and Ant Design

## Tech Stack

- **Backend**: Java, Spring Boot, Spring Data JPA
- **Frontend**: React, JavaScript, Ant Design
- **Database**: PostgreSQL
- **Cloud**: Google Cloud Storage
- **Features**: JWT auth, geocoding, location search
