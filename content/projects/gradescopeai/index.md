An educational platform with AI-powered grading using RAG pipeline, enabling instructors to manage courses and provide intelligent feedback.

<!--more-->

## Features

- **Course Management**: Instructors can manage courses, assignments, and submissions
- **Private Invitation System**: Students join courses via private invitation tokens
- **AI-Powered Grading**: High-performance grading microservice integrating Google Gemini via LangChain
- **RAG Pipeline**: Improved feedback quality using ChromaDB with 120+ curated IELTS essays and rubrics
- **Document Management**: Integrated Google Cloud Storage for asset delivery
- **Modern Frontend**: React/TypeScript with Material-UI, Redux Toolkit, and React Router

## Tech Stack

- **Backend**: Java/Spring Boot (main service), Python/FastAPI (grading microservice)
- **Frontend**: React, TypeScript, Material-UI, Redux Toolkit
- **Database**: PostgreSQL, ChromaDB (vector store)
- **AI**: Google Gemini, LangChain, RAG architecture
- **Cloud**: Google Cloud Storage
