A social media application with AI-assisted image generation powered by OpenAI DALL-E and robust search functionality.

<!--more-->

## Features

- **AI Image Generation**: Integrated OpenAI DALL-E for AI-assisted image creation
- **Secure Authentication**: Token-based authentication with JWT
- **Scalable Backend**: Built in Go with gorilla/mux for HTTP routing
- **Powerful Search**: Elasticsearch for robust, efficient search across posts and personal content
- **Cloud Deployment**: Deployed to Google App Engine for scalability
- **Responsive UI**: Material-UI with React Photo Album and Yet Another React Lightbox

## Tech Stack

- **Backend**: Go, gorilla/mux
- **Frontend**: React, JavaScript, Material-UI
- **Search**: Elasticsearch
- **AI**: OpenAI DALL-E
- **Cloud**: Google App Engine
